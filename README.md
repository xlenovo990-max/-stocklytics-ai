# Stocklytics AI 📊

An AI-powered, privacy-first inventory & profit forecasting assistant tailored for modern e-commerce sellers and Indie Hackers.

---

## 🔥 Enterprise-Grade Features

* **⚡ BYOK (Bring Your Own Key) Architecture**: Fully powered by OpenRouter (DeepSeek-Chat Engine). Pay only for what you actually use.
* **🔒 Stateless Security (Malware-Proof)**: Based on our zero-trace privacy protocol, this application never stores your API key in localStorage or any external servers. The key lives strictly within your browser's runtime memory (RAM) and evaporates instantly the moment you close the tab or if your device crashes/reboots.
* **🌐 Instant Bilingual Toggle**: Seamlessly switch between English and Traditional Chinese (Hong Kong/Taiwan) UI layouts and AI report generation.
* **📊 Smart E-commerce CFO Prompting**: Generates production-ready financial metrics including restock triggers, capital efficiency analysis, net margins, and estimated ROI calculations based on your supplier cost and target retail price.

---

## 🛠️ Tech Stack & Architecture

This project is built with the philosophy of extreme minimalism and ultimate security:
* **Frontend**: Semantic HTML5 & Modern CSS3 Dashboard UI.
* **Core Engine**: Vanilla JavaScript (ES6+) asynchronous state machines.
* **Zero Dependencies**: No third-party tracking scripts, cookies, or database connections. 100% auditable and open-source.

---

## 🚀 How to Run Locally

Since this app is completely client-side and serverless, you can deploy or run it anywhere in 2 seconds:

1. Clone this repository.
2. Open index.html directly in any modern browser (Chrome, Safari, Brave, Edge).
3. Input your OpenRouter API Key and start calculating your margins with peace of mind!
